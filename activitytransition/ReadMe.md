# Android中Activity的转场动画

## demo目的
	- 了解、熟悉android5.0以下的转场动画用法，以及android5.0以上转场动画新特性
	- android5.0转场动画的兼容与适配处理

## 用到的技术
	- 掌握rotate、translation、alpha和scale补间动画的xml文件
	- 掌握android5.0中分解、滑动、淡入淡出以及共享元素4种新转场动画的用法，学会使用代码和xml控制
